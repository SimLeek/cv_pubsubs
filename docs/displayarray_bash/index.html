displayarray cli¶

DisplayArray. Display NumPy arrays.

Usage:

displayarray (-w <webcam-number> | -v <video-filename> | -t <topic-name>[,dtype])… [-m <msg-backend>] displayarray -h displayarray –version

Options:
-h, --help

Show this help text.

--version

Show version number.

-w <webcam-number>, --webcam=<webcam-number>

Display video from a webcam.

-v <video-filename>, --video=<video-filename>

Display frames from a video file.

-t <topic-name>, --topic=<topic-name>

Display frames from a topic using the chosen message broker.

-m <msg-backend>, --message-backend <msg-backend>

Choose message broker backend. [Default: ROS] Currently supported: ROS, ZeroMQ

--ros

Use ROS as the backend message broker.

--zeromq

Use ZeroMQ as the backend message broker.
